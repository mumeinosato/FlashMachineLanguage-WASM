# フラッシュ機械語 レベル仕様書（改訂版）

## 基本仕様（共通）
- 四則演算のうち **ADD / SUB のみ使用可能**
- 扱う値は **int32 範囲**
- 実際の機械語は x86_64（REX.W 使用可能）
- 即値は常に **32bit（imm32）**
- mov/add/sub/xor のみが使用可能
- レジスタは RAX/RBX/RCX/RDX（64bit）を使用
- 終了命令なし：バイト列が尽きたら終了
- 初期値はすべて 0
- int32 範囲を超えた場合は -1 として扱う
- メモリアクセスは不可
- 計算回数 = ADD/SUB 命令の「数」

---

# レベル別ルール

## **Level 1**
- 扱う数値：**実際に使う数値が int8 範囲（-128〜127）**
- 即値は32bitだが、**値の範囲が ±128 内**
- 計算回数：**1 回（ADD or SUB のみ）**

例：
- `mov eax, 0x65`
- `sub eax, 0x26`

---

## **Level 2**
- 扱う数値：**int16 範囲（±32767）**
- 計算回数：**1 回のみ**

---

## **Level 3**
- 扱う数値：**int32 範囲全部**
- 計算回数：**1 回のみ**

---

## **Level 4（旧 Level4+Level5 統合版）**
- 扱う数値：**int32 の全範囲**
- 計算回数：**複数回の ADD / SUB を許可**
- レジスタ演算も自由（複数回使用可）
